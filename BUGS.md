
|***Баг-репорт-001***|Невозможность получить объявление по ID сразу после создания|
|--------------------|--------------------------|
|**Приоритет:**|High|
|**Описание:**|После успешного создания объявления вызов  GET/api/1/item/{id} возвращает 404 или пустой объект|
|**Шаги воспроизведения:**|POST /api/1/item- получить ID, GET /api/1/item/{ID}|
|**Ожидаемый результат:**|Должны вернуть созданное объявление по ID|
|**Фактический результат:**|Возвращает 404|
