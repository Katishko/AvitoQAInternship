
Host: configurable (по умолчанию https://qa-internship.avito.com)</br>

В API есть 4 основных ручки:</br>
- POST /api/1/item         — создать объявление</br>
- GET  /api/1/item/:id     — получить объявление по id</br>
- GET  /api/1/:sellerID/item — получить все объявления продавца</br>
- GET  /api/1/statistic/:id — получить статистику по объявлению</br>
(и аналогичные пути под /api/2/...; в тестах используется /api/1)</br>



|**ID**|TC-001 |
|--------------|---------------------------|
|**Заголовок:**|Создание объявления с корректным телом запроса|
|**Описание:**|baseUrl доступен|
|**Приоритет:**|High|
|**Шаги воспроизведения:**|написать запрос POST /api/1/item   {</br>"sellerID": 123456,</br>"name": "Test item",</br> "price": 100,</br> "statistics": {"likes": 1, "viewCount": 2, </br>"contacts": 3}</br> } и получить ответ| 
|**Ожидаемый результат:**| HTTP 200,тело ответа содержит поля: id (string), sellerId (int), name (string), price (int), statistics (with likes/viewCount/contacts), createdAt (string)|
|||
</br>
</br>


|**ID**|TC-002 |
|--------------|--------------|
|**Заголовок:**|Получение объявления по ID|
|**Описание:**|Создано объявление|
|**Приоритет:**|High|
|**Шаги воспроизведения:**|запрос GET /api/1/item/{id},|
|**Ожидаемый результат:**|Возвращает HTTP 200, тело ответа содержит массив или объект с объявлением, у которого id соответствует запросу|
|||


</br>

|**ID**|TC-003 |
|--------------|--------------------|

|**Описание:**|У продавца есть >=1 объявление|

|**Ожидаемый результат:**|Возвращает  HTTP 200,тело ответа — массив; как минимум одно объявление с sellerId == sellerID|
