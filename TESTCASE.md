
Host: configurable (по умолчанию https://qa-internship.avito.com)

В API есть 4 основных ручки:
- POST /api/1/item         — создать объявление
- GET  /api/1/item/:id     — получить объявление по id
- GET  /api/1/:sellerID/item — получить все объявления продавца
- GET  /api/1/statistic/:id — получить статистику по объявлению
(и аналогичные пути под /api/2/...; в тестах используется /api/1)



|**ID**|TC-001 |
|--------------|---------------------------|
|**Заголовок:**|Create item - valid payload|
|**Описание:**|baseUrl доступен|
|**Приоритет:**|High|
|**Шаги воспроизведения:**|написать запрос POST /api/1/item   {</br>"sellerID": 123456,</br>"name": "Test item",</br> "price": 100,</br> "statistics": {"likes": 1, "viewCount": 2, </br>"contacts": 3}</br> } и получить ответ| 
|**Ожидаемый результат:**| HTTP 200,тело ответа содержит поля: id (string), sellerId (int), name (string), price (int), statistics (with likes/viewCount/contacts), createdAt (string)|
|||



|**ID**|TC-002 |
